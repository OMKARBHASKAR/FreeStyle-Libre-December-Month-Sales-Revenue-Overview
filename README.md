# 📊 FreeStyle Libre — December Sales & Revenue Dashboard

## Overview
This project analyzes **December sales performance of FreeStyle Libre products** using a **Power BI dashboard**.  
The dashboard converts transactional sales data into clear visual metrics to help understand **revenue performance, profit trends, and sales executive contribution**.

### Key Metrics
| Metric | Value |
|------|------|
| Quantity Sold | 574 |
| Total Revenue | ₹1,997,133 |
| Total Profit | ₹848K |
| Profit Margin | 42.45% |
| Transactions | 300 |

---

## Dashboard Highlights

- **Revenue & Customer Trend** – Daily tracking of revenue and customer activity across the month  
- **Sales Executive Performance** – Revenue comparison between Priya, Rahul, Anita, and Kiran  
- **Payment Mode Distribution** – Order share across UPI, Bank Transfer, Cash, and Card  
- **Selling vs Cost Price Analysis** – Profitability comparison across executives  
- **Cumulative Growth Tracking** – Profit and quantity progression throughout the month
